# Security Policy

## Responsible disclosure
If you believe you have found a security issue, please report it privately.

- Email: security@YOURDOMAIN.com
- Include: a clear description, reproduction steps, and impact assessment

## Expectations
- Do not disclose the issue publicly before we respond
- Do not test against production data or real merchant stores without permission

## Bounty policy
We do not offer a guaranteed bug bounty or compensation.

## Scope
This repository is documentation-only and contains no production code or secrets.
