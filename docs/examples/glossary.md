# Glossary

## Additional Scripts
Legacy Shopify mechanism for injecting scripts into the Thank You and Order Status pages.

## ScriptTags
Legacy Shopify mechanism for injecting scripts into storefront pages.

## Web Pixel
Shopify's sandboxed pixel environment for customer events.

## App Pixel
A Web Pixel extension installed by a Shopify app.

## CAPI
Conversions API, a server-side events interface used by some ad platforms.

## Measurement Protocol
Server-side events interface for Google Analytics 4.

## De-duplication
Process of removing duplicate events using a stable event_id.

## Consent gating
Policy that restricts identifiers and data fields based on user consent state.
