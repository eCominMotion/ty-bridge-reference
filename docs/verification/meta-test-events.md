# Meta Test Events

Use Meta Events Manager to confirm that server-side events are received.

## Steps

1) Open Events Manager and select your Pixel.

2) Go to Test Events.

3) Start a test session for your domain.

4) Trigger a checkout flow from Shopify.

5) Confirm that the event appears with the expected event_id.

## Notes

- Test Events may show processing warnings that do not appear in production reporting.

- If the event does not appear, check consent state, pixel ID, and destination configuration.
